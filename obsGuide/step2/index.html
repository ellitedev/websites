Watch out!

Make sure your have OBS Studio installed! If you don't have it installed, check out this page.
In this tutorial we will be using Windows 10, it should be very similair on Mac and Linux.

On a fresh install of OBS, you should be greated by this:

Welcome to the setup wizard!
On this page, make sure Optimize for streaming, recording is secondary is selected, and click Next

You should now be greated by this page.

Now select 1280x720 from the dropdown list.

And select 60 in the FPS dropdown menu and click Next.

You should now be greated by this page.

In the Service dropdown list, select Server input field.
Custom...

The page should now look like this.

Click Yes.

Watch out!

The information provided in the images may be out of date! Use the information provided in the text below!

Now enter rtmp://[region].ingest.vrcdn.live/relay in the Server input field.
And enter your Stream Key given by @Ellite#7242 or @TheWay123#1756 in the Stream Key input field.

Make sure that Prefer hardware encoding and Estimate bitrate with bandwidth test (may take a few minutes) are checked.

Don't worry!

You can still continue with the tutorial if you don't have a Stream Key yet. Just enter any random text instead, and replace it with your Stream Key when you get it later in the settings.

It should now look like this.

Click Apply Settings.

You should now be back on this page! Click the button below to continue to the next step.
